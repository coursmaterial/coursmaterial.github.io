Kapitel 3

Wat schafft Dir?
Wat schaffs du?

Ech schaffen als Verkeeferin.

Wat sidd Dir vu Beruff?
Wat bass du vu Beruff?

Ech sinn Informatiker.

Wou schafft Dir?
Wou schaffs du?

Ech schaffen op engem Büro.

Wéini schafft Dir?
Wéini schaffs du?

Ech schaffe vun aacht Auer moies bis hallwer sechs nomëttes.

Wéi vill Auer ass et?

Et ass zwanzeg op zéng moies.

Wou schafft Dir?
  • Ech schaffen an + Gebaier.
    Ech schaffen an engem Buttek / an enger Schoul / an enger Klinick.
  • Ech schaffen bei + Persounen.
    Ech schaffe bei engem Dokter / bei engem Bäcker / bei enger Coiffeuse.
  • Ech schaffen op + Administratiounen a Plazen.
    Ech schaffen op engem Büro / op enger Bank / op engem Maart / op enger Tankstell.
